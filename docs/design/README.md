# Portfolio Design Documentation

This directory contains the maintainable design reference for the portfolio visual system.

- `portfolio-visual-direction.md` is the canonical visual specification for issue #61.
- `current-ui-audit.md` records the pre-migration state and baseline validation results.
- `implementation-map.md` maps design-system concerns to the files that implement them.

New pages and components should follow the same semantic tokens, typography roles, spacing rhythm, component primitives, light/dark theme behavior, and accessibility expectations. Validate changes in light mode, dark mode, mobile widths, desktop widths, keyboard navigation, and reduced-motion mode before shipping.
